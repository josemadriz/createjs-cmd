# createjs-cmd

The module of Createjs used CMD standard. And you can use npm to install it.

## Module versions
- EaselJS: 0.8.2
- TweenJS: 0.6.2
- SoundJS: 0.6.2
- PreloadJS: 0.6.2

## Install

```shell
npm install --save createjs-cmd
```

## Usage

```javascript
import createjs from 'createjs-cmd';
```
